# EAISI ProcessMining

<!-- badges: start -->

<!-- badges: end -->

The goal of EAISI ProcessMining is to apply the process mining
techniques learner at Eindhoven Artificial Intelligence Systems
Institute (EAISI).

These techniques were shown using a free licence of Celonis. This
repository aims to apply the same using [bupaR](https://bupar.net/)
package in R based on SAP data.
